# CurrenSee 💰

**CurrenSee** is a Flutter-based currency companion app that provides real-time exchange rates, currency conversion, rate alerts, market news, and a full user account system — all wrapped in a clean gold-themed UI with dark mode support.

> 4th Semester E-Project

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Flutter (Dart) |
| Authentication | Firebase Auth — Email/Password & Google Sign-In |
| Database | Cloud Firestore |
| Notifications | Firebase Cloud Messaging + flutter_local_notifications |
| State Management | Provider |
| HTTP | http package |
| Local Storage | shared_preferences |
| PDF Export | pdf + printing |

---

## Features

### 🔐 Authentication
- Email/Password Sign Up & Sign In
- Google Sign-In
- Onboarding screen shown only on first launch
- Auth state persistence — auto-login on relaunch

### 💱 Currency Tools
- **Currency Converter** — Convert between currencies in real time
- **Currency List** — Browse all supported currencies
- **Exchange Rate Information** — View live rates at a glance
- **Conversion History** — Track past conversions

### 🔔 Rate Alerts
- Automated background rate checks every **15 minutes**
- Local push notifications when rates change for: USD, EUR, GBP, SAR, AED, CNY (vs PKR)
- Toggleable via master switch and per-feature toggle in User Preferences

### 📰 News & Trends
- Currency News and Market Trends screen
- News Detail view with full article content
- URL launcher for external sources

### 👤 User Account
- Profile screen with editable display name and photo (via image picker)
- Edit Profile and Change Password screens
- Firestore-backed user data

### 💬 Community
- Feedback submission (stored in Firestore)
- All Feedbacks / Reviews screen (read community feedback)

### ⚙️ Settings & More
- **Dark / Light Mode** toggle (persisted via Provider + ThemeMode)
- **User Preferences** screen
- **Notifications** management screen
- **User Support & Help Center**
- PDF export support (printing package)

## Authentication Rules

### Sign Up — Password Requirements

Password must contain:
- At least **8 characters**
- At least one **uppercase letter** (A–Z)
- At least one **lowercase letter** (a–z)
- At least one **number** (0–9)
- At least one **special character** (`!@#$%^&*` etc.)

### Sign In — Error Messaging

Sign-in failures show specific, user-friendly messages:

| Error | Message Shown |
|---|---|
| Invalid email format | "Please enter a valid email address." |
| Wrong email or password | "Incorrect email or password." |
| Disabled account | "Your account has been disabled." |
| Too many attempts | "Too many attempts. Please try again later." |
| No internet | "No internet connection. Please check your network." |
| Google Sign-In cancelled | "Google sign-in was cancelled." |

```bash
flutterfire configure
```

### Required Firebase Services

- Firebase Authentication (Email/Password + Google Sign-In)
- Cloud Firestore
- Firebase Cloud Messaging (FCM)

---

## Notifications

The app uses two notification layers:

| Type | Usage |
|---|---|
| **FCM (Firebase Cloud Messaging)** | Remote push notifications from backend |
| **flutter_local_notifications** | Local rate-change alerts triggered in-app |

Notifications are delivered through the `high_importance_channel` on Android. Rate alerts fire automatically every **15 minutes** in the background via `RateCheckService`, comparing current rates against the previously saved snapshot in `shared_preferences`.

## Currency API

Live exchange rates are fetched from a free, open-source API — no API key required:

```
https://cdn.jsdelivr.net/npm/@fawazahmed0/currency-api@latest/v1/currencies/pkr.json
```
